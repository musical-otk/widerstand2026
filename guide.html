🏠 홈
  • 다음 공연 D-day + 캐스팅·좌석
  • 예정 일정 목록 (설정에서 표시 배역 선택)
  • 아이드 도서카드 현황 / 쿠폰 잔여
  • 최근 관람 이력 + 마일스톤 달성 표시
📅 캘린더
  • 배역 탭 → 배우 선택 → 해당 회차 필터
  • ✅ = 기록한 회차
  • 날짜 클릭 → 회차 목록 + 기록 추가/상세
🎫 쿠폰·도장
  • 50% 실물 할인권 잔여(현재) / 예정(최종)
  • 4칸 달성 → 할인권 또는 폴라로이드 선택
  • 7칸 달성 → 폴라로이드 📸 (캐스트 지정)
  • 이력 패널에서 달성 날짜·혜택 변경
😌 통계
  • 히트맵 — 좌석 방문 횟수 (1층·2층)
  • 배우별 — 역할별 관람 횟수
  • 페어별 — 조합 선택 후 조회, 행 클릭 시 날짜 목록
  • 지출 — 총 지출 / 평균 티켓가
📝 팁
스케줄 연동 — 날짜 클릭 → 회차 옆 "+ 기록"으로 캐스팅 자동 입력
아이드 도서카드 — 기록 저장 시 도장판 선택하면 자동 도장. 7칸 초과 시 새 카드 자동 생성
사진 첨부 — 원본 화질 저장. 탭: 전체화면 / 두 번 탭: 가로 꽉채움
데이터 백업 — 설정 → JSON 내보내기/가져오기. 사진 포함 옵션 있음
